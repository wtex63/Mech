# Snapmaker A350 Design Rules

Store shared Fusion design rule files (.dru) used across multiple A350 workflows.

## File Naming

- [tool]-[material]-v[major].[minor].dru

Examples:

- shared-pla-v1.0.dru
- shared-acrylic-v1.1.dru

## Notes

Use the process-specific folders when a .dru only applies to one workflow.
