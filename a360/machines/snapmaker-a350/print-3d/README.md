# Snapmaker A350 3D Print

Use this folder for 3D print profile notes and validation logs.

## Track Here

- Filament type and nozzle size
- Layer height and wall settings
- Supports and adhesion settings
- Temperature and speed ranges

## Output Checks

- Confirm orientation for strength and surface finish.
- Confirm expected clearances from design parameters.
- Confirm estimated print time and material usage.
