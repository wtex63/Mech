# Design Review Checklist

## Fit And Function

- [ ] Primary requirements are traceable to design features.
- [ ] Motion ranges and collision checks are complete.
- [ ] Interfaces match mating components.

## Manufacturability

- [ ] Material selection is documented.
- [ ] Tool access and fixturing are feasible.
- [ ] Tolerances are realistic for process capability.

## CAD Quality

- [ ] Timeline is clean and understandable.
- [ ] Sketches are constrained.
- [ ] Component names follow naming conventions.

## Documentation

- [ ] Critical dimensions identified.
- [ ] Exports generated (STEP, STL, drawings as needed).
- [ ] Revision note added.
