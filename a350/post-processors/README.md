# A350 Post Processors

Store CAM post files and notes in this directory.

## Contents

- .cps files
- machine-specific setup notes
- verification output samples

## Naming

Use machine and controller identifiers in file names:

[machine]-[controller]-v[major].cps

Example:

haas-vf2-fanuc-v1.cps

Snapmaker A350 example:

snapmaker-a350-generic-grbl-v1.cps
