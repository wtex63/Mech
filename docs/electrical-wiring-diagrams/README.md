# Electrical Wiring Diagrams

Use this folder for electrical wiring diagrams and supporting standards.

## Suggested Structure

- standards/: Naming, symbols, and drawing conventions
- diagrams/: Source files and exported PDFs
- revisions/: Change notes per wiring release

## Naming Convention

- wiring-[system]-v[major]-[minor].pdf
- wiring-[system]-source-v[major]-[minor].[ext]

## Recommended Metadata

Each diagram should document:

- Voltage rails
- Connector and pin mapping
- Wire gauge assumptions
- Revision date and author
